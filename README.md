# Youtube Search

## Running the project

### 1. Using Docker

1. Add Google API keys in the .env file
2. Run Following commands

```
docker-compose build
docker-compose up
```

### 2. Local Installation

1. Exports env file
2. Install MongoDB
3. Install Node Js
4. Run following commands

```
npm install
node index
```
